# student-api
## Features
- Add student (POST)
- View all students (GET)
- View single student (GET by ID)
- Update student (PUT)
- Delete student (DELETE)

## Tech Used
- Node.js
- HTTP module
- Postman (for testing)

## How to run
1. Install Node.js
2. Run: node server.js
3. Open Postman and test endpoints
